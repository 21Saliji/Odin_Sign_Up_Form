# Odin Sign-Up Form

This is a simple sign-up form project created as part of The Odin Project curriculum. It demonstrates basic HTML form structure, CSS styling, and JavaScript form validation.

## Features

- **Form Fields**: First Name, Last Name, Email, Phone Number, Password, and Confirm Password
- **Validation**:
  - Password must be at least 8 characters long
  - Confirm Password must match the Password field
- **Responsive Design**: Styled with CSS for a clean, modern look
- **Odin Branding**: Includes the Odin logo and themed styling

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## How to Use

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Fill out the form and submit to test validation

## Project Structure

- `index.html`: Main HTML file containing the form
- `style.css`: CSS styles for layout and design
- `script.js`: JavaScript for form validation logic
- `images/`: Directory containing the Odin logo image
- `fonts/`: Directory for custom fonts

## Note

This is not a real online service - it's a demonstration project for learning purposes. The form does not actually submit data anywhere.
